# Sparse Polynomial Zonotope

## Definition

$$
PZ := \left\{
\sum_{i=1}^{h} ( \prod_{k=1}^{p} \alpha_{k}^{E_{k,i}} ) G_{\dot, i} +
\sum_{j=1}^{q} \beta_{j} G_{I(\cdot, j)} |
\alpha_{k}, \beta_{j} \in [-1,1]
\right\}
$$
