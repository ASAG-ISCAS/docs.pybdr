# Sparse Polynomial Zonotope

## Definition

$$
PZ := \\left\\{
\sum_{i=1}^{h} \( \prod\_{k=1}^{p} \alpha\_{k}^{E\_{k,i}} \) G\_{\dot, i} +
\sum\_{j=1}^{q} \beta\_{j} G\_{I(\dot, j)} |
\alpha\_{k}, \beta\_{j} \in [-1,1]
\\right\\}
$$
