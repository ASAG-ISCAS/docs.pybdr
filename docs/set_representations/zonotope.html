Zonotope

Definition​

Z≔{c+∑i=1pβigi ∣ βi∈[−1,1]}{\cal Z} \coloneqq \{ c+\sum_{i=1}^{p} \beta_{i} g^{i} \ \big| \ \beta_{i} \in [-1,1] \}Z:={c+i=1∑p​βi​gi ∣∣​ βi​∈[−1,1]}

shorthand as Z=<c,G>{\cal Z}=\left<c,G \right>Z=⟨c,G⟩, order defined as o=GcolGrowo=\frac{G_{col}}{G_{row}}o=Grow​Gcol​​

Example​

from pyrat.geometry import Zonotope
from pyrat.geometry.operation import partition
from pyrat.util.visualization import plot

# random zonotope with 100 generators in 2d space
z = Zonotope.rand(2, 100)
# split this zonotope in smaller cells as zonotope
zono_parts = partition(z, 1, Geometry.TYPE.ZONOTOPE)

# visualize the final result
plot([*zono_parts,z], [0, 1])

Arithmetic​

addition or '+'​

  • with another zonotope
<c0,G0>+<c1,G1>=<c0+c0,[G0 G1]>\left<c_{0},G_{0} \right> + \left<c_{1},G_{1} \right> = \left<c_{0}+c_{0},\left[G_{0} \ G_{1} \right] \right>⟨c0​,G0​⟩+⟨c1​,G1​⟩=⟨c0​+c0​,[G0​ G1​]⟩
  • with another interval
<c,G>+[a,b]=<c+a+b2,[G,b−a2]>\left< c,G \right> + [a,b] = \left<c+\frac{a+b}{2},\left[G,\frac{b-a}{2} \right] \right>⟨c,G⟩+[a,b]=⟨c+2a+b​,[G,2b−a​]⟩
  • with another real number
<c,G>+a=<c+a,G>\left<c,G \right> + a = \left< c+a,G \right>⟨c,G⟩+a=⟨c+a,G⟩

multiplication or '*'​

  • with another real number
<c,G>⋅a=<a⋅c,a⋅G>\left<c,G \right> \cdot a = \left< a \cdot c , a \cdot G\right>⟨c,G⟩⋅a=⟨a⋅c,a⋅G⟩
  • with another interval
<c,G>⋅[a,b]=<a+b2c,[b−a2c,a+b2G,b−a2G]>\left< c,G \right> \cdot [a,b] = \left< \frac{a+b}{2} c, \left[ \frac{b-a}{2} c, \frac{a+b}{2} G, \frac{b-a}{2} G \right] \right>⟨c,G⟩⋅[a,b]=⟨2a+b​c,[2b−a​c,2a+b​G,2b−a​G]⟩
  • with another zonotope
<c0,G0>⋅<c1,G1>=<c0c1,[c1G0,c0G1,g00g10,⋯ ,g0ig1j,⋯ ,g0pg1q]>\left<c_{0},G_{0} \right> \cdot \left<c_{1},G_{1} \right> = \left< c_0 c_1, \left[ c_1 G_0, c_0 G_1, g_0^0 g_1^0, \cdots, g_0^i g_1^j, \cdots, g_0^p g_1^q \right] \right>⟨c0​,G0​⟩⋅⟨c1​,G1​⟩=⟨c0​c1​,[c1​G0​,c0​G1​,g00​g10​,⋯,g0i​g1j​,⋯,g0p​g1q​]⟩

matrix multiplication or '@'​

  • real matrix with zonotope matrix
(X[Z])ij=∑k=1nXik[Z]kj(X[Z])_{ij} = \sum_{k=1}^{n} X_{ik} [Z]_{kj}(X[Z])ij​=k=1∑n​Xik​[Z]kj​
  • zonotope matrix with real matrix
([X]Z)ij=∑k=1n[X]ikZkj([X]Z)_{ij} = \sum_{k=1}^{n} [X]_{ik} Z_{kj}([X]Z)ij​=k=1∑n​[X]ik​Zkj​
  • zonotope matrix with another zonotope matrix
([X][Z])ij=∑k=1n[X]ik[Z]kj([X][Z])_{ij} = \sum_{k=1}^{n} [X]_{ik} [Z]_{kj}([X][Z])ij​=k=1∑n​[X]ik​[Z]kj​

where [X]⊆Ro×n[X] \sube \R^{o \times n}[X]⊆Ro×n and [Z]⊆Rn×p[Z] \sube \R^{n \times p}[Z]⊆Rn×p
