# Taylor Model

## Definition

$$
{\cal T} \coloneqq \left\{
P(x)+y \ \big| \ y \in I
\right\}
$$

where $P: \mathbb{R}^{p} \rightarrow \mathbb{R}^{n}$ is a polynomial function and $I \subset \mathbb{R}^{n}$ is an
interval.
