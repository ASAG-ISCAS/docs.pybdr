# Taylor Model

## Definition

$$
T(x) = \left\{
p(x)+y | y \in I
\right\}
$$

where $p: \mathbb{R}^{p} \rightarrow \mathbb{R}^{n}$ is a polynomial function and $I \subset \mathbb{R}^{n}$ is an interval.
