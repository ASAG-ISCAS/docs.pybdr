# Polytope

## Definition

### H-Representation

$$
P := \\{ x \in \mathbb{R}^{n} | Cx \leq d\\}, C \in \mathbb{R}^{q \times n}, d \in \mathbb{R}^{q}
$$

### V-Representation

$$
P := \\{
\sum_{i=1}^{r} {\alpha}\_{i} v^{i} | {\alpha}\_{i} \in \mathbb{R}, {\alpha}\_{i} \geq 0, \sum\_{i=1}^{r} {\alpha}\_{i}
=1
\\}
$$
