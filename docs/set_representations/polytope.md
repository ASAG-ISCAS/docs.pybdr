# Polytope

## Definition

### H-Representation

$$
{\cal P} \coloneqq \{ x \in \mathbb{R}^{n} | Cx \leq d\}, C \in \mathbb{R}^{q \times n}, d \in \mathbb{R}^{q}
$$

### V-Representation

$$
{\cal P} \coloneqq \{
\sum_{i=1}^{r} {\alpha}_{i} v^{i} | {\alpha}_{i} \in \mathbb{R}, {\alpha}_{i} \geq 0, \sum_{i=1}^{r} {\alpha}_{i}
=1
\}
$$
