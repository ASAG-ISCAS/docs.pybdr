PyBDR

Boundary analysis based Reachability analysis Toolbox for dynamic systems in Python

Theory

The boundary of the reachable set of given initial set at specified time instant is equal to the reachable set of the boundary of the initial set at this given time instant.

Data Structure

Interval arithmetic is supported by means of tensor, compatible with the broadcast mechanism.

Computation

Easy to take partial derivatives of arbitrary order with support of interval arithmetic.
